# dhamma-scraper

Script to find out when the applications open.
Configured for the test date (27 Jan - 07 Feb). Dates can be changed on line 9 in main.py.

## Install requrements
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
deactivate
```

## Run the script
```
source env/bin/activate
python main.py
deactivate
```
